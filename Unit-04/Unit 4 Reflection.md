# Unit 4 Reflection

[In this unit, I started learning and using the list function in HTML. By learning ordered lists and unordered lists, I realized that the content on a webpage should not be placed randomly. Instead, it needs to be planned and organized in a logical and structured way. The list function helps me arrange information on a webpage more clearly and neatly, making it easier for users to understand the content.
Before this, I thought website design was just about putting text, images, and links onto a page. However, after using lists, I realized that website design is also about how to organize information. For example, when a webpage includes several items, steps, features, or categories, using a list can make the content look cleaner and more organized. It can also help highlight the important points. This helped me take another step forward in understanding webpage planning and design.
At the same time, the list function also showed me that HTML is a programming language with strong readability. Many HTML tags are easy to understand, such as <ul>, <ol>, and <li>. Even beginners can roughly understand what these tags are used for. Through these tags, we can not only create webpage content, but also clearly show the relationship between different pieces of information.
Overall, this unit helped me understand that HTML is not only a tool for displaying content. It can also help us organize the structure of a webpage in a more systematic way. Although the list function seems simple, it is very useful in website design and helped me develop a deeper understanding of front-end web design.]
