# Unit 3 Reflection

[During this activity of collecting website design requirements, I experienced a feeling that was very different from my previous understanding of Computer Science. In my impression, Computer Science was always a subject that mainly involved dealing with machines. I thought it was mostly about writing code, solving programming problems, and working with computers, software, and technical tools. I used to believe that as long as the code worked correctly and the website could be built, the task was completed.
However, through this activity, I realized for the first time that website design is not only about working with machines. It also requires communication with real people. We need to understand the client’s ideas, interests, needs, and the personal style they want to show. Only through communication can we know how the website should be designed and what colors, content, and layout would best meet the client’s expectations.
This helped me understand that Computer Science is not only about technology. It also includes the ability to understand others, listen carefully, and solve real-world problems. A good website should not only function properly, but also truly fit the person who will use it. This experience gave me a more complete understanding of website design and helped me realize that Computer Science is also closely connected to communication between people.]
